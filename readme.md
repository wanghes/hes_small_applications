## 项目简介
这是一个个人博客小程序版，使用 [Mpvue](https://github.com/Meituan-Dianping/mpvue) 编写而成，服务端使用的是Bmob后端云，无需开发服务端接口即可使用，快速便捷，适合个人使用，功能包括签到，收藏文章，查看文章，消息通知，评论文章，回复评论等。欢迎提建议和意见。
